# 单片机计数器原理

1. 单片机的一个计数周期时 1 us，2^8 代表在计数未溢出时一共可以记录 65536 us（0 ~ 65535）
   
   换算一下就是 65.536 ms

2. TMOD是八位寄存器，在赋值时只能通过 0x -- 的方式一次性赋值，不能单独给某一个赋值

3. 八位的前四个时T1方式字段，后四位时T0方式字段

4. 两种字段的排列方式均为
   
   a.GATE
   
   b.C / T
   
   c.M1
   
   d.M0

5. #### 置初值：使用定时器T0 / T1
   
   ``` 
   TH0 = 0x3c;
   TL0 = 0xb0;
   或者
   TH0 = （65535 - 50000） / 256；
   TH0 = （65535 - 50000） % 256；
   //其中50000是自定义的，可以更改成其他的值，这里的50000是指定时50ms
   //如果需要使用定时器T1，只需要将TH0 更改成 -> TH1 <- 即可
   ```
   
   

6. #### 启动定时器：
   
   a.定时器的启动是在另外一个东西里实现，那个是 TCON ，TCON不像是 TMOD 必须要八位一次性赋值，它可以进行单独的赋值，在学习时只需要记住每个位的名字即可
   
   从左到右分别是 // D7 D6 ................................ D0
   
   从左到右分别是//TF1  -   TR1  -   TF0  -  TR0   -  IE1  -  IT1  -  IE0  -  IT0  （八个可以独立赋值的位）
   
   b.定时器的启动则是输入 TR0 = 1；停止则是输入 TR0 = 0；
   
   c.以上所提到的 TR0 实际上是寄存器 TCON 里八个位中的一个位，用来控制定时器的启动或关闭
   
   

7. #### 定时器的溢出
   
   当定时器计数到最大值 ，最大值为2^8 时，会将 TCON 中的溢出标志位 TF0 置为1（默认位0）
   
   在使用定时器的时候经常需要将定时器手动置为 0，来保证循环可持续性。或者说当定时器计数到了最大值时，可以通过 --- while(TF0 == 0); --- 来跳出循环
   
   

8. #### 重置初值（定时器的循环）
   
   受限于单片机自身溢出前只能记录很短的一段时间，所以可以使用循环来延长它的计数时间，并且在一个循环完成后，需要将计数器的初值重置，重新开始计数
   
       TH0 = 0x3c;
       TL0 = 0xb0;
       或者
       TH0 = （65535 - 50000） / 256；
       TH0 = （65535 - 50000） % 256；
   
    

9. #### 清溢出
   
   在计数器溢出后，需要手动的将溢出标志位清零
   
   即 ----->  TF0 = 0;  <-------
